Benchmark

Please note that the 8mb benchmark file has not been loaded with the page. This is to reduce bandwidth. It will load once (and hopefully cache ;) when you start running the first test.

You can choose to bench a specific parser or to run them all in random order. Note that in the beginning, these have not been optimized for this purpose. I've tried to call them with the minimum overhead possible. Expect improvements in the future, if their authors want to participate of course. Have a parser you want to add? Just let me know.
